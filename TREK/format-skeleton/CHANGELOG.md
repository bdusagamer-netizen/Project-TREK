# Changelog

All notable changes to Trek will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-04

### Added
- Initial release of Trek programming language
- Complete language syntax specification
- TextMate grammar for VS Code syntax highlighting
- JSON Schema for AST validation
- VS Code language configuration and editor support
- Comprehensive documentation and examples
- Installation guides for multiple platforms
- Support for visual element declaration (shapes, colors, positioning)
- Variable definitions and type system
- Functions and classes
- Control flow (if/else, for, while loops)
- Module system (import/export)
- Declarative positioning system
- Background and drawing element support

### Features
- ✨ Visual-first programming syntax
- 📦 Modular code organization
- 🎨 Built-in drawing and styling
- 🔄 Functional programming support
- 💾 Multiple data types (numbers, strings, booleans, hex colors)
- 🎯 Flexible positioning system

### Documentation
- Language specification with full syntax reference
- Multiple code examples and patterns
- Installation instructions for different platforms
- VS Code extension setup guide
